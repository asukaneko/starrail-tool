# 《崩坏：星穹铁道》伤害计算乘区完全指南（4.0版本）

## 1. 伤害计算乘区概述

伤害计算乘区是指最终算出伤害的各个乘区，我总结为**角色区**和**怪物区**。

**角色区**因为出伤方式不同导致有多种算法，其中包括：
- **直伤**（常规攻击伤害）
- **击破**（含超击破）
- **持续伤害**（以下简称dot）
- **欢愉伤害**（4.0版本新增）

**怪物区**的结算方式对所有伤害类型都一样，因此怪物区也叫**上三区**（实际为四区），本篇先从怪物区开始介绍。

---

## 2. 怪物伤害乘区

怪物区有四个乘区，分别是：**防御、抗性、易伤、减伤**，彼此之间为**乘算**关系。

**怪物区公式：**
$$\text{怪物区} = \text{防御区} \times \text{抗性区} \times \text{易伤区} \times \text{减伤区}$$

### 2.1 防御区

防御区计算公式：
$$\text{防御区} = \frac{200 + 10 \times \text{角色等级}}{200 + 10 \times \text{角色等级} + \text{怪物防御} \times (1 - \text{减防率})}$$

**关键机制：**
- 减防存在**上限100%**，超出部分无效
- 当减防达到100%时，防御区记为**1**（即无视防御）
- 常见减防来源：银狼（大招45%+天赋12%）、佩拉（大招40%）、量子套（20%）、虚无命途（18%）等

**举例：**
二魂银狼携带「谎言随风飘荡」，释放终结技给有量子弱点的怪物施加所有减防debuff：
- 银狼大招：45%
- 银狼天赋：12%
- 光锥「谎言随风飘荡」：24%
- 量子套：20%
- **总计：101%** → 实际生效100%，防御区记为**1**

如果没有这些减防：
- 防御区约为**0.46**（假设80级角色对80级怪物）

### 2.2 抗性区

$$\text{抗性区} = 1 - \text{最终抗性} = 1 - \max(0, \text{基础抗性} - \text{减抗})$$

**关键机制：**
- 有弱点时基础抗性为**0**，无弱点时为**20%**
- 减抗**超出**基础抗性后，最终抗性记为**0**，**不会转化为易伤**
- 抗性可以减到负数（部分角色/光锥），此时伤害提高

**举例：**
银狼给怪物添加量子弱点并触发天赋：
- 基础抗性：20%（无弱点时）
- 添加弱点后：0%
- 银狼天赋减抗：13%
- **最终抗性：max(0, 0% - 13%) = 0%** → 抗性区记为**1**

**注意：** 如果银狼对已有量子弱点的怪物使用，则基础抗性为0%，减抗13%后最终抗性为0%，抗性区仍为1（不会变成1.13）。

### 2.3 易伤区

$$\text{易伤区} = 1 + \text{易伤百分比}$$

**举例：**
二魂银狼自带20%易伤，易伤区记为**1.2**。

### 2.4 减伤区

减伤区通常由怪物技能或环境效果提供，部分角色/光锥有**减伤穿透**能力，但较为稀有，常规情况下视为**1**。

### 2.5 怪物区综合对比

| 场景 | 防御区 | 抗性区 | 易伤区 | 减伤区 | 怪物区总计 |
|------|--------|--------|--------|--------|-----------|
| 二魂银狼全debuff | 1 | 1 | 1.2 | 1 | **1.2** |
| 无减防减抗 | 0.46 | 0.8 | 1 | 1 | **0.368** |

**伤害提升倍数：** 1.2 / 0.368 ≈ **3.26倍**（原文3.68倍计算有误）

## 3. 角色区

角色区目前有**四种**算法：直伤、击破、dot、欢愉伤害。

### 3.1 直伤伤害计算乘区

直伤区有四个乘区：**攻击、倍率、双暴、增伤**。

$$\text{直伤} = \text{攻击区} \times \text{倍率区} \times \text{双暴区} \times \text{增伤区}$$

**举例：**
大黑塔携带智识队友，用强化战技攻击解读层数为42层的主目标：

- **攻击区：** 基础白值2600 × (1 + 1.15装备攻击 + 0.8终结技自拐) = **5070**
- **倍率区：** 对主目标倍率**952%**（9.52）
- **双暴区：** 假设面板100%/200%，双暴区 = **2**
- **增伤区：** 强化战技自拐50%增伤 = **1.5**

**计算：**
$$5070 \times 9.52 \times 2 \times 1.5 = 144,799.2 \times \text{怪物区}$$

### 3.2 击破伤害计算乘区

击破伤害有四个乘区：**击破系数区、属性系数区、韧性系数区、击破特攻区**。

$$\text{击破伤害} = \text{击破系数} \times \text{属性系数} \times \text{韧性系数} \times (1 + \text{击破特攻})$$

| 乘区 | 说明 |
|------|------|
| **击破系数** | 仅与角色等级有关，80级为**3767.55** |
| **属性系数** | 物理/火=1，风=1，雷=2，冰=0.5，量子=0.5，虚数=0.5 |
| **韧性系数** | (怪物韧性长度 + 20) / 40 |
| **击破特攻** | 角色面板属性 |

**等级压制修正：**
$$\text{实际击破系数} = \text{基础系数} \times \frac{\text{角色等级} + 20}{\text{怪物等级} + 20}$$

**举例：**
80级流萤击破380韧性条的火属性怪物，击破特攻400%：
- 击破系数：3767.55
- 属性系数：2（火）
- 韧性系数：(380 + 20) / 40 = 10
- 击破特攻区：1 + 4 = 5

**计算：** 3767.55 × 2 × 10 × 5 = **376,755** × 怪物区

**忘归人虚韧性击破：** 与常规击破计算方式相同，但韧性条长度按虚韧性计算。

### 3.3 超击破伤害计算乘区

超击破有四个乘区：**击破系数区、击破倍率区、削韧区、击破特攻区**。

$$\text{超击破伤害} = \text{击破系数} \times \text{击破倍率} \times \text{削韧区} \times (1 + \text{击破特攻})$$

| 乘区 | 说明 |
|------|------|
| **击破系数** | 同击破伤害，80级3767.55 |
| **击破倍率** | 角色转化率，流萤50%、同协主100%，可叠加 |
| **削韧区** | **实际削韧值**（非削韧值/10），但单次上限**10点** |
| **击破特攻** | 角色面板属性 |

**关键机制：**
- 阮·梅的击破效率提升**实际削韧值**，但**不影响10点上限**
- 忘归人的超击破倍率可以叠加

### 3.4 dot伤害计算乘区

#### 3.4.1 角色dot伤害

三个乘区：**攻击、倍率、增伤**（**无暴区**）。

$$\text{dot伤害} = \text{攻击区} \times \text{倍率区} \times \text{增伤区}$$

#### 3.4.2 击破dot伤害

三个乘区：**击破系数区、击破特攻区、属性dot区**。

| 属性 | 属性dot区 | 特殊机制 |
|------|----------|---------|
| **火** | 1 | 基础dot |
| **雷** | 2 | 触电伤害 |
| **风** | 风化层数 | 最多叠加5层 |
| **物理** | 裂伤 | 有百分比上限 |

**物理击破dot（裂伤）完整公式：**

对精英怪/Boss：
$$\text{裂伤伤害} = \min\left(7\% \times \text{最大生命值}, 2 \times \text{基础击破伤害}\right) \times (1 + \text{击破特攻}) \times \text{怪物区}$$

对小怪：
$$\text{裂伤伤害} = \min\left(16\% \times \text{最大生命值}, 2 \times \text{基础击破伤害}\right) \times (1 + \text{击破特攻}) \times \text{怪物区}$$

**注意：** 原文漏掉了**2×基础击破伤害**这个上限，高击破特攻时此上限会生效。

**冰/量子击破：**
- **冰击破**：附加伤害，**可以暴击**，属于直伤
- **量子击破**：延迟伤害（纠缠），**可以暴击**，属于直伤
- 两者**不属于dot伤害**

### 3.5 欢愉伤害计算乘区（4.0版本新增）

欢愉伤害是4.0版本新增的特殊伤害类型，机制独特。

$$\text{欢愉伤害} = \text{基础值} \times \text{倍率} \times \text{双暴区} \times (1 + \text{笑点}) \times (1 + \text{欢愉度}) \times (1 + \text{增笑度}) \times \text{怪物区（防御/抗性/易伤/真伤）}$$

| 乘区 | 说明 | 数值来源 |
|------|------|---------|
| **基础值** | 固定值，80级为**7535** | 角色等级 |
| **倍率** | 技能描述的欢愉伤害倍率 | 技能本身 |
| **双暴区** | (1 + 暴伤) | 角色面板 |
| **笑点区** | (1 + 当前笑点) | 战技点上方显示，全队共享 |
| **欢愉度** | (1 + 欢愉度%) | 角色面板新属性，类似增伤 |
| **增笑度** | (1 + 增笑度%) | 稀有独立乘区 |
| **怪物区** | 防御×抗性×易伤×真伤 | **注意：不吃增伤** |

**关键机制：**

1. **笑点与好活当赏：**
   - **阿哈时刻**：阿哈行动时消耗所有笑点，触发全队欢愉技，伤害基于**当前笑点**
   - **好活当赏**：阿哈时刻结束后，消耗的笑点转化为角色持续2回合的buff，伤害基于**好活层数**
   - **阿哈时刻外**：伤害基于好活当赏的笑点数

2. **不吃攻击力、不吃增伤**：属性球、增伤辅助对欢愉伤害无效

3. **吃真伤加成**：真实伤害加成可以提高欢愉伤害

4. **欢愉度来源：**
   - 基础欢愉度（角色自带）
   - 爻光：速度转欢愉度（速度≥120时+30%，每超1点+1%，最多200点）
   - 火花：攻击力转欢愉度

**培养要点：**
- ✅ 堆：双暴、速度（转欢愉度）、抗性穿透、减防、击破特攻（部分角色）
- ❌ 不堆：攻击力、增伤

## 真实伤害的核心特点

真实伤害是3.0版本新增的特殊伤害机制，与普通伤害有本质区别：

1. **无视防御**：不受敌人防御力影响
2. **无属性伤害**：不受属性抗性影响（既不是物理也不是元素伤害）
3. **不吃常规乘区**：不受增伤、暴伤、易伤、减防等任何乘区加成

## 计算公式

**通用公式：**
$$真实伤害 = 固定数值来源 \times 真实伤害倍率$$

**固定数值来源**根据技能描述不同，可以是：
- **原伤害**（基于你原本造成的伤害）
- **目标生命值上限**（基于敌人的最大生命值）
- 其他固定数值

## 实战举例

以记忆命途开拓者（Remembrance Trailblazer）为例：

假设你的飞霄（Feixiao）：
- 攻击力：3,500
- 终结技倍率：700%
- 真实伤害倍率：30%（来自记忆主）

**计算过程：**
1. 基础伤害 = 3,500 × 700% = 24,500
2. 真实伤害 = 24,500 × 30% = **7,350**

这7,350点真实伤害会**直接扣除**，无视敌人的防御和抗性。

## 关键要点

| 特性 | 说明 |
|------|------|
| **独立计算** | 真实伤害在普通伤害计算完成后单独附加 |
| **不可暴击** | 真实伤害本身不会暴击，基于原伤害计算 |
| **不吃辅助** | 不受停云、布洛妮娅等角色的增伤/加攻影响 |
| **固定值** | 只要原伤害确定，真伤数值就是固定的 |

## 战略价值

真实伤害的优势在于：
- **高防御敌人克星**：面对高防BOSS时，普通伤害被减免很多，但真伤全额生效
- **稳定性**：不受随机因素（暴击、抗性波动）影响，输出可预期
- **辅助友好**：不需要投资增伤/暴伤辅助，节省资源

目前主要通过**记忆命途开拓者**为队友提供真实伤害能力，适合搭配高频输出角色使用。

## 欢愉伤害的核心特点

欢愉伤害是4.0版本新增的特殊伤害类型，与真实伤害和普通伤害都有区别：

| 特性 | 说明 |
|------|------|
| **不吃攻击力** | 不基于角色攻击力计算 |
| **不吃增伤** | 不受"伤害提高"类效果影响（如属性球、增伤辅助） |
| **可以暴击** | 受双暴区影响 |
| **吃防御/抗性/易伤** | 受减防、抗性穿透、易伤影响 |
| **吃真实伤害** | 可以吃到真实伤害加成 |

## 计算公式

**完整公式：**
$$欢愉伤害 = 基础乘区 \times 倍率区 \times 双暴区 \times (1 + 笑点区) \times 欢愉度 \times 增笑度 \times 抗性区 \times 防御区 \times 易伤区 \times 真伤区$$

**各乘区详解：**

| 乘区 | 说明 | 数值来源 |
|------|------|----------|
| **基础乘区** | 固定值，约7535（80级） | 角色等级决定，类似超击破 |
| **倍率区** | 技能描述的欢愉伤害倍率 | 技能本身 |
| **双暴区** | (1 + 暴伤) | 角色面板 |
| **笑点区** | (1 + 笑点数值) | 战技点上方显示的笑点 |
| **欢愉度** | (1 + 欢愉度%) | 角色面板新属性，类似增伤 |
| **增笑度** | (1 + 增笑度%) | 稀有乘区，独立增伤 |
| **抗性/防御/易伤/真伤** | 与传统伤害相同 | 辅助提供 |

## 关键机制解析

### 1. 笑点（Punchline）与好活当赏（Certified Banger）

- **笑点**：战技点上方显示，全队共享，通过角色技能获得
- **阿哈时刻**：当阿哈行动时，消耗所有笑点触发全队欢愉技
- **好活当赏**：阿哈时刻结束后，将消耗的笑点转化为角色持续2回合的"好活"层数（显示在角色头像下方）

**重要区别：**
- **阿哈时刻内**：欢愉伤害基于**当前笑点**计算
- **阿哈时刻外**：欢愉伤害基于**好活当赏**的笑点数计算

### 2. 欢愉度（Elation）

- 每个欢愉角色有基础欢愉度
- 可通过角色机制提升：
  - **爻光**：速度转欢愉度（速度≥120时+30%，每超1点+1%，最多200点）
  - **火花**：攻击力转欢愉度
- 欢愉度上限通常为230%

### 3. 欢愉技（Elation Skill）

- 欢愉角色特有技能，在阿哈时刻按**参演编号**顺序释放
- 造成欢愉伤害，伤害基于当前笑点/好活数值

## 实战举例

假设80级爻光：
- 基础乘区：7535
- 技能倍率：200%
- 暴击伤害：150%
- 当前笑点：50
- 欢愉度：150%
- 增笑度：20%
- 敌人防御减免后：0.5
- 抗性穿透后：0.9
- 易伤：1.3
- 真伤加成：1.1

**计算：**
$$7535 \times 2.0 \times 2.5 \times (1+50) \times 2.5 \times 1.2 \times 0.9 \times 0.5 \times 1.3 \times 1.1$$

## 配队与培养要点

1. **多欢愉角色**：欢愉角色越多，阿哈速度越快，出伤频率越高
2. **高速角色**：加快笑点积累，提升阿哈速度
3. **不需要堆**：攻击力、增伤（属性球无用）
4. **需要堆**：双暴、速度（转欢愉度）、抗性穿透、减防
5. **适合辅助**：缇宝、赛飞儿（提供减防/易伤/抗性穿透，不提供增伤）

## 与真实伤害的区别

| 对比项 | 欢愉伤害 | 真实伤害 |
|--------|----------|----------|
| **基础数值** | 固定7535（等级相关） | 基于原伤害或生命上限 |
| **双暴** | ✅ 吃双暴 | ❌ 不吃双暴 |
| **防御/抗性** | ✅ 吃减防/穿透 | ❌ 无视防御抗性 |
| **增伤** | ❌ 不吃增伤 | ❌ 不吃增伤 |
